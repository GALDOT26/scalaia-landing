Volver a SCALA OS

Politica de privacidad

Version preliminar comercial - 2026-06-09

1. Responsable y alcance

Esta politica describe como SCALA OS trata informacion vinculada a clientes, usuarios, sellers, operaciones ecommerce e integraciones conectadas al servicio.

SCALA OS funciona como una solucion tecnologica llave en mano para centralizar datos, automatizaciones y analitica operativa. El tratamiento de datos se realiza para prestar el servicio contratado, configurar integraciones, brindar soporte, mejorar la experiencia y cumplir obligaciones legales.

2. Datos que podemos tratar

Segun el alcance contratado, SCALA OS puede tratar datos de identificacion del cliente, datos de usuarios autorizados, datos operativos, datos de integraciones, metricas comerciales, reportes, consultas de soporte e informacion necesaria para configurar y mantener el servicio.

SCALA OS no solicita datos innecesarios para el servicio y recomienda no cargar informacion sensible que no sea requerida para la operacion.

3. Finalidades

Los datos pueden utilizarse para crear y administrar la cuenta del cliente, configurar integraciones, mostrar tableros y reportes, generar recomendaciones, brindar soporte, mejorar seguridad y cumplir obligaciones legales.

4. Integraciones con terceros

SCALA OS puede conectarse con plataformas externas como Mercado Libre, Mercado Pago, tiendas online, proveedores de IA, servicios de comunicacion, facturacion, analitica o infraestructura. Cada integracion opera bajo sus propias politicas, permisos y terminos.

SCALA OS no vende datos personales ni informacion operativa del cliente a terceros.

5. Inteligencia artificial

SCALA OS puede usar modelos de inteligencia artificial para generar analisis, recomendaciones, respuestas, contenido, clasificaciones y automatizaciones. Cuando se utilicen proveedores externos de IA, se enviara solo la informacion necesaria para completar la tarea, procurando minimizar datos sensibles.

6. Seguridad

SCALA OS aplica medidas razonables de seguridad para proteger la informacion, tales como control de accesos, permisos por usuario, resguardo de credenciales, registros de actividad, buenas practicas de infraestructura y separacion de entornos segun corresponda.

7. Conservacion

Los datos se conservaran mientras sean necesarios para prestar el servicio, cumplir el contrato, resolver incidencias, conservar respaldos, auditar acciones o cumplir obligaciones legales. Al finalizar la relacion, el cliente puede solicitar baja, exportacion o eliminacion razonable de datos, sujeto a limites tecnicos, contractuales o legales.

8. Derechos del titular

Cuando corresponda, los titulares de datos pueden solicitar acceso, actualizacion, rectificacion, eliminacion u oposicion al tratamiento de sus datos personales, conforme la normativa aplicable.

9. Confidencialidad

SCALA OS tratara la informacion comercial, operativa y tecnica del cliente como confidencial. El acceso interno a datos se limita a quienes lo necesiten para prestar soporte, mantener el servicio, resolver incidentes o cumplir tareas autorizadas.

10. Cambios en esta politica

SCALA OS podra actualizar esta politica para reflejar mejoras del servicio, cambios regulatorios, nuevas integraciones o ajustes operativos. La version vigente sera la publicada o informada al cliente por los canales correspondientes.

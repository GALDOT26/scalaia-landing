Terminos y condiciones

Version preliminar comercial - 2026-06-09

1. Identificacion del servicio

SCALA OS es una solucion tecnologica llave en mano para sellers de ecommerce, orientada a centralizar operacion, datos, automatizaciones y analitica comercial en canales como Mercado Libre, Mercado Pago, tiendas online y herramientas administrativas compatibles.

El servicio puede incluir software, configuracion inicial, integraciones, capacitacion, tableros, automatizaciones, soporte operativo y acompanamiento de adopcion, segun el alcance acordado con cada cliente.

2. Alcance

El alcance exacto se define en una propuesta comercial o acuerdo de implementacion. Podra incluir configuracion de cuenta, conexion con canales autorizados por el cliente, gestion de publicaciones, stock, precios, rentabilidad, Mercado Ads, respuestas asistidas, tableros, metricas, automatizaciones, capacitacion y seguimiento de uso.

SCALA OS no reemplaza la responsabilidad comercial, fiscal, contable, legal ni operativa del seller. El cliente conserva la decision final sobre precios, publicaciones, promociones, respuestas, facturacion y cumplimiento de politicas de cada plataforma.

3. Uso autorizado

El cliente declara que cuenta con autorizacion suficiente para conectar sus cuentas, datos, usuarios, canales y credenciales al servicio. El cliente se compromete a usar SCALA OS de forma licita, responsable y compatible con las politicas de Mercado Libre, Mercado Pago, AFIP/ARCA, proveedores logisticos y cualquier otra plataforma conectada.

Queda prohibido usar el servicio para manipular datos, reputacion, reviews, compradores o competidores de forma indebida; compartir datos de contacto donde las politicas de la plataforma lo prohiban; publicar informacion falsa; vulnerar derechos de terceros; o intentar acceder a datos o cuentas sin autorizacion.

4. Integraciones y autorizaciones

Las integraciones se realizan mediante credenciales, APIs, OAuth, tokens o mecanismos disponibles para cada plataforma. El cliente puede revocar accesos cuando corresponda desde la plataforma externa o solicitando la baja tecnica.

SCALA OS podra ejecutar acciones automatizadas solo cuando el cliente las haya habilitado expresamente o cuando formen parte del alcance contratado. Las acciones sensibles pueden requerir aprobacion humana, configuracion de umbrales o validacion previa.

5. IA, recomendaciones y decision final

SCALA OS puede utilizar inteligencia artificial para analizar datos, generar sugerencias, redactar contenido, detectar riesgos y asistir decisiones. Las recomendaciones de IA son herramientas de apoyo y no garantizan resultados comerciales especificos.

6. Precios y modalidad comercial

El servicio se comercializa bajo modalidad empresarial a medida. El valor depende de cantidad de cuentas, volumen operativo, modulos activos, integraciones requeridas, nivel de soporte, automatizaciones y alcance de implementacion.

7. Soporte y disponibilidad

SCALA OS procurara mantener el servicio operativo y brindar soporte razonable segun el plan contratado. Pueden existir interrupciones por mantenimiento, incidentes tecnicos, caidas de terceros, cambios de APIs, limites de plataformas, problemas de conectividad o fuerza mayor.

8. Datos, confidencialidad y seguridad

SCALA OS tratara la informacion del cliente de forma confidencial y solo para prestar el servicio, mejorar su funcionamiento, brindar soporte, cumplir obligaciones legales o ejecutar instrucciones del cliente.

El cliente recibe un derecho de uso limitado, no exclusivo y no transferible durante la vigencia del servicio contratado.

10. Limitacion de responsabilidad

SCALA OS no sera responsable por perdidas indirectas, lucro cesante, sanciones de plataformas, decisiones comerciales del cliente, errores de carga, cambios de politicas externas, caidas de servicios de terceros, uso indebido del sistema o acciones ejecutadas sin respetar recomendaciones, alertas o aprobaciones configuradas.
